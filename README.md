# Global Commodity Shocks, Trade Linkages, and Economic Resilience

## Project Overview
This project investigates how global commodity shocks (energy, food, metals) transmit through India's international trade networks to affect domestic economic stability.

## Project Structure
- **data/**: Raw and processed datasets
- **networks/**: Network graph objects and metrics
- **src/**: Source code for all analysis
- **notebooks/**: Jupyter notebooks for exploration
- **models/**: Saved trained models
- **outputs/**: All figures, tables, and reports
- **docs/**: Documentation and data dictionaries
- **presentations/**: Presentation slides
- **reports/**: Written reports

## Setup Instructions
1. Install Python 3.8+
2. Run: `pip install -r requirements.txt`
3. Download data using scripts in `src/data_collection/`
4. Run analysis scripts in order
